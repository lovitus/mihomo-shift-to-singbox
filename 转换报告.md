### ⚠️ 有变化/限制的转换

| 原始配置 | sing-box 处理方式 | 说明 |
|----------|-------------------|------|
| Hysteria v1 port hopping (`ports: "33891,65520-65530"`) | 仅使用第一个端口 (33891) | sing-box hysteria v1 不原生支持端口跳跃 |
| `dialer-proxy` | 转换为 `detour` 字段 | 功能等效 |
| `fallback` 代理组类型 | 转换为 `urltest` | sing-box 无原生 fallback,urltest 近似等效(按延迟选择而非按顺序优先) |
| `ip-version: ipv4-prefer` | 未转换 | sing-box 通过 DNS strategy 控制,已在 DNS 配置中设置 `prefer_ipv4` |
| `PROCESS-NAME-REGEX` 规则 | 转换为 `process_path_regex` | sing-box 路由规则不支持 process_name_regex,使用 path_regex 替代 |
| `DOMAIN-REGEX` 规则 | 直接转换为 `domain_regex` | ✅ 完全支持 |
| `GEOIP,lan` | 转换为 `ip_is_private: true` | sing-box 原生支持 |
| `AND/OR/NOT` 逻辑规则 | ⚠️ 未转换(已跳过) | sing-box 支持 logical rules 但格式不同,涉及的规则很少 |
| `.mrs` 格式规则集 (reject2, reject3) | 使用 YAML 替代源 + Go 转换器 | mihomo 二进制格式不兼容,改用 YAML 源文件 |

## 如何运行

### 1. 启动规则转换器 (必须先启动)
```bash
cd rule-converter
go build -o rule-converter.exe .
./rule-converter.exe
# 监听 127.0.0.1:8081
```

### 2. 启动 sing-box (测试模式)
```bash
# 确保 sing-box 安装了 quic 和 clash_api 支持
go install -tags "with_quic,with_clash_api" github.com/sagernet/sing-box/cmd/sing-box@latest

# 运行 (使用 17893 端口避免与 mihomo 冲突)
sing-box run -c singbox-config.json
```

### 3. 重新生成配置 (如修改了原始 mihomo 配置)
```bash
python convert_to_singbox.py
```

### 4. 切换到生产模式 (启用 TUN)
编辑 `convert_to_singbox.py` 第 738 行:
```python
inbounds = build_inbounds(testing_mode=False)
```
然后重新生成并以管理员权限运行。

## Fallback 行为说明

sing-box **没有**原生的 `fallback` 代理组类型。Clash/Mihomo 的 `fallback` 按用户定义的顺序尝试节点,第一个可用的就使用。sing-box 的 `urltest` 则按延迟选择最快的节点。

| | Clash `fallback` | sing-box `urltest` |
|---|---|---|
| 选择逻辑 | 按顺序,第一个可用 | 延迟最低 |
| 切换行为 | 当前节点不可用才切换 | 可能频繁切换到更快节点 |
| 可预测性 | 高(顺序固定) | 中(取决于网络状况) |

**缓解方案**: 设置较高的 `tolerance` 值(如 5000ms),可以减少不必要的切换,但不能完全模拟有序故障转移。

## 已知限制

1. **Proxy Provider 不支持**: sing-box 官方版不支持 proxy-provider(订阅链接)。4个使用 provider 的代理组仅有 DIRECT 占位
2. **Hysteria v1 端口跳跃**: 仅使用第一个端口,不支持多端口跳跃
3. **Fallback 行为差异**: sing-box urltest 按延迟选择,而 mihomo fallback 按顺序优先(见上方详细说明)
4. **AND/OR/NOT 逻辑规则**: 2条逻辑复合规则未转换(影响很小)
5. **节点不可达**: 因 __CLASSMASK_* 占位符,所有节点实际不可连接,但配置加载正常
6. **block outbound (仅Python版)**: 代理组引用 block 作为可选成员,无法移除,产生1个deprecation警告
